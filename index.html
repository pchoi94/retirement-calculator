Monte Carlo Simulation i

Monte Carlo note: The simulation uses paired historical return/inflation observations but does not yet preserve multi-year market regimes with block bootstrap or model full asset-class allocation.
Historical mode: Monte Carlo uses 10,000 simulations and samples annual stock returns from historical S&P 500 total-return data from 1928–2024. User-entered volatility, return floor, return cap, and trial count are not used.
